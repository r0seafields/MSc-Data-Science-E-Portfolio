Unit 7 – Artificial Neural Networks (ANNs)

Machine Learning Module

Overview

This unit introduced Artificial Neural Networks (ANNs) and explored how they can be used to solve classification problems. Through practical notebook activities, I investigated how perceptrons make decisions, how weights influence predictions, and how multi-layer perceptrons use hidden layers and activation functions to learn more complex patterns from data.

Practical Activity

I completed two practical notebook exercises that explored the implementation of a single-layer perceptron and a multi-layer perceptron. These activities provided practical experience of how neural networks learn and classify data.

  • Experiment 2 – Perceptron AND Operator (PDF)
  • Experiment 3 – Multi-Layer Perceptron (PDF)

Reflection

This was one of the most engaging units in the module because it introduced the principles behind modern Artificial Intelligence. Unlike traditional statistical methods, neural networks learn patterns directly from data, allowing them to solve increasingly complex classification problems.

The perceptron exercise demonstrated how a single artificial neuron can implement simple logical functions such as the AND operator, highlighting the importance of weights and thresholds in determining predictions. The multi-layer perceptron activity then extended these concepts by introducing hidden layers and sigmoid activation functions, illustrating how additional network complexity enables more sophisticated pattern recognition.

Completing these practical activities improved my understanding of how neural networks are trained and provided a strong foundation for studying more advanced deep learning techniques, including convolutional neural networks later in the module.

Skills Developed

  • Understanding Artificial Neural Network architecture.
  • Implementing perceptrons for binary classification.
  • Understanding weights, biases and activation functions.
  • Developing practical machine learning skills using Python and Jupyter Notebooks.
  • Interpreting neural network outputs and classification performance.

Summary

This unit provided an introduction to the fundamental concepts of Artificial Neural Networks and demonstrated how increasingly complex network architectures improve learning and classification performance. The notebook activities strengthened my understanding of neural network design and provided valuable practical experience using Python to implement machine learning algorithms.
