Unit 11 – Model Performance Measurement

Machine Learning Module

Overview

This unit explored how the performance of machine learning models can be evaluated using a range of classification and regression metrics. Through practical experimentation, I investigated how different evaluation measures provide insight into model accuracy, reliability and predictive performance, reinforcing the importance of selecting appropriate metrics when assessing machine learning models.

Practical Activity

I completed a practical notebook investigating a variety of model performance metrics, including confusion matrices, precision, recall, F1 score, ROC curves, Area Under the Curve (AUC), Root Mean Squared Error (RMSE), Mean Absolute Error (MAE) and the coefficient of determination (R²). The activity also explored the effect of adjusting Logistic Regression parameters on model performance.

Reflection

This unit demonstrated that building a machine learning model is only one part of the overall process. Equally important is evaluating how well a model performs using appropriate performance measures. Working through the notebook helped me understand that no single metric is suitable for every problem, and that different evaluation measures provide different perspectives on model performance.

I found the comparison between classification and regression metrics particularly valuable. Understanding when to use measures such as precision, recall and F1 score, rather than relying solely on accuracy, highlighted the importance of considering the context of a machine learning problem. Experimenting with Logistic Regression parameters also reinforced how model configuration can influence predictive performance and generalisation.

Overall, this unit strengthened my ability to critically evaluate machine learning models and reinforced that selecting suitable evaluation metrics is essential for developing reliable and trustworthy Artificial Intelligence systems.

Skills Developed

  • Evaluating classification and regression models.
  • Interpreting confusion matrices and performance metrics.
  • Understanding precision, recall, F1 score and ROC/AUC.
  • Applying RMSE, MAE and R² to regression problems.
  • Investigating the effect of model parameter tuning.
  • Developing critical evaluation skills for machine learning models.

Summary

This unit consolidated my understanding of model evaluation by demonstrating how different performance metrics provide complementary insights into machine learning models. Completing the practical activity strengthened my ability to assess both classification and regression algorithms, ensuring that models are evaluated fairly and appropriately for the task they are intended to perform.
