Unit 11 – Model Performance Measurement

This unit focused on evaluating the performance of machine learning models using a range of classification and regression metrics. Through the practical notebook activities, I explored confusion matrices, accuracy, precision, recall, F1 scores, ROC curves, Area Under the Curve (AUC), Root Mean Squared Error (RMSE), Mean Absolute Error (MAE) and the coefficient of determination (R²). Understanding these evaluation techniques demonstrated that building a model is only part of the machine learning process; selecting appropriate performance metrics is equally important when determining how well a model performs.

I also investigated how changing Logistic Regression parameters influenced model performance. This reinforced the importance of selecting suitable model settings rather than relying on default parameters, as even small adjustments can affect prediction quality and model generalisation.

Reflection

This unit changed the way I think about machine learning models. Before completing these activities, I tended to focus primarily on building a model and obtaining predictions. However, I now understand that evaluating a model is just as important because different performance measures reveal different aspects of model behaviour.

Working with confusion matrices helped me understand how predictions can be broken down into true positives, true negatives, false positives and false negatives. From these values, I could see how metrics such as precision, recall and the F1 score provide a much more detailed assessment than overall accuracy alone, particularly when working with imbalanced datasets.

I also found the experiments involving ROC curves and AUC particularly useful. Modifying the Logistic Regression parameters demonstrated that changing values such as the regularisation parameter (C) and solver produced only small differences in AUC, but it reinforced the importance of parameter tuning when seeking the best performing model. Similarly, experimenting with different prediction values showed how dramatically the R² score changes depending on prediction quality, highlighting the relationship between model accuracy and evaluation metrics.

Looking back, this unit reinforced that model evaluation is an essential stage of the machine learning lifecycle. Rather than accepting model outputs at face value, I now appreciate the importance of selecting appropriate metrics to determine whether a model is genuinely reliable and suitable for real-world applications.
